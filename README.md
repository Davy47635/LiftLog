## App Features
- Register, login, logout and delete profile
- Log workouts with category, duration, date and notes
- Add and delete targets with progress tracking
- Insights screen with bar chart and stats
- Search and filter workouts by text, category and date
- Streak tracking on home screen
- Dark/light mode toggle
- CSV export
